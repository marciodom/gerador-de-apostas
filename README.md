# Gerador de Apostas (PWA)

 Aplicação Web desenvolvida como Progressive Web App (PWA) para análise estatística de dados de sorteios da Loteria Estadual, publicada via GitHub Pages e com suporte a funcionamento offline por meio de Service Worker.
 O sistema foi criado para auxiliar usuários na visualização de padrões históricos, frequência de números e organização de combinações para apostas.


🚀 Demonstração

🔗 Acesse:
https://marciodom.github.io/gerador-de-apostas/


🎯 Funcionalidades

• Análise de sorteios passados
• Identificação de frequência de números
• Visualização de padrões históricos
• Geração de combinações para apostas
• Funcionamento offline
• Instalação como aplicativo no celular (PWA)
